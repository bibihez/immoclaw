# Email Relances Génériques — FR

## Relance certificateur électrique (J+5 après inspection)

```
Objet : Rapport de contrôle électrique — {adresse}

Bonjour,

Suite au contrôle de l'installation électrique effectué le {date_inspection} au {adresse}, pourriez-vous me transmettre le rapport dès qu'il sera disponible ?

La mise en vente est en cours et ce document est requis pour le dossier.

{signature_agent}
```

## Relance attestation de sol (J+14)

```
Objet : Relance — Attestation de sol {adresse}

Bonjour,

Je me permets de revenir vers vous concernant ma demande d'attestation de sol pour le bien au {adresse}, introduite le {date_demande}.

Pourriez-vous me donner un délai pour la réception de ce document ?

{signature_agent}
```

## Relance notaire (pré-compromis)

```
Objet : Dossier compromis — {adresse}

Cher Maître,

Je me permets de vous relancer concernant le dossier de compromis pour le bien au {adresse}.

Acheteur : {nom_acheteur}
Vendeur : {nom_vendeur}
Prix convenu : {prix} EUR

Tous les documents sont disponibles dans le dossier Drive partagé. Pourriez-vous me confirmer une date pour la signature du compromis ?

{signature_agent}
```

## Relance acheteur (financement, J+30 après compromis)

```
Objet : Suivi financement — {adresse}

Bonjour {prenom_acheteur},

Dans le cadre de l'achat du bien au {adresse}, pourriez-vous me communiquer l'état d'avancement de votre demande de prêt hypothécaire ?

Le compromis prévoit un délai de {delai_jours} jours pour l'obtention du crédit.

Je reste à votre disposition.

{signature_agent}
```

## Relance générique (prestataire qui ne répond pas)

```
Objet : Relance — {objet_original}

Bonjour,

Je me permets de revenir vers vous concernant {description_demande} pour le bien au {adresse}.

N'ayant pas reçu de retour, je souhaitais savoir si vous pouvez traiter cette demande. Dans le cas contraire, merci de me le signaler.

{signature_agent}
```
