# Email Certificateur PEB — FR

## Demande initiale

```
Objet : Demande de certificat PEB — {adresse}

Bonjour,

Dans le cadre de la mise en vente du bien situé au {adresse}, {code_postal} {commune}, je souhaite faire réaliser un certificat PEB.

Détails du bien :
- Type : {type} ({nb_chambres} chambres, {surface}m²)
- Année de construction : {year_built}
- Étage : {etage si applicable}

Pourriez-vous me communiquer vos disponibilités et votre tarif pour cette intervention ?

Merci d'avance.

{signature_agent}

Réf. mandat : {mandate_ref}
```

## Relance J+3

```
Objet : Relance — Demande PEB {adresse}

Bonjour,

Je me permets de revenir vers vous concernant ma demande de certificat PEB pour le bien au {adresse}.

Avez-vous la possibilité d'effectuer cette intervention prochainement ?

{signature_agent}
```

## Relance J+7

```
Objet : 2e relance — Certificat PEB {adresse}

Bonjour,

N'ayant pas reçu de retour concernant ma demande de certificat PEB pour le bien au {adresse}, je me permets de vous relancer.

Si vous n'êtes pas disponible, merci de me le signaler afin que je puisse me tourner vers un autre certificateur.

{signature_agent}
```

## Post-RDV : demande de rapport

```
Objet : Rapport PEB — {adresse}

Bonjour,

Suite à votre visite du {date_rdv} pour le bien au {adresse}, pourriez-vous me transmettre le certificat PEB dès qu'il sera disponible ?

Merci d'avance.

{signature_agent}
```
