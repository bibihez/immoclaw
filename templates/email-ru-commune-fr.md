# Email Renseignements Urbanistiques Commune — FR

## Demande initiale

```
Objet : Demande de renseignements urbanistiques — {adresse}

Madame, Monsieur,

Dans le cadre de la mise en vente du bien situé au {adresse}, {code_postal} {commune}, je souhaite obtenir les renseignements urbanistiques relatifs à ce bien.

Données cadastrales :
- Adresse : {adresse}
- Code postal : {code_postal}
- Division : {division}
- Section : {section}
- Numéro parcelle : {parcelle}
- CaPaKey : {capakey}

Propriétaire : {nom_vendeur}

Merci de me communiquer le délai de traitement et les éventuels frais.

{signature_agent}

Réf. mandat : {mandate_ref}
```

## Relance J+30

```
Objet : Relance — Renseignements urbanistiques {adresse}

Madame, Monsieur,

Je me permets de revenir vers vous concernant ma demande de renseignements urbanistiques pour le bien au {adresse}, introduite le {date_demande}.

Pourriez-vous me communiquer l'état d'avancement de cette demande ?

{signature_agent}
```

## Relance J+45

```
Objet : 2e relance — Renseignements urbanistiques {adresse}

Madame, Monsieur,

N'ayant pas encore reçu les renseignements urbanistiques pour le bien au {adresse} (demande du {date_demande}), je me permets de vous relancer.

Ce document est indispensable à la constitution du dossier de vente. Pourriez-vous me donner un délai précis ?

{signature_agent}
```

## Relance J+60

```
Objet : 3e relance URGENTE — Renseignements urbanistiques {adresse}

Madame, Monsieur,

Cela fait maintenant 60 jours que ma demande de renseignements urbanistiques pour le bien au {adresse} a été introduite (le {date_demande}).

Ce document bloque la finalisation du dossier de vente. Je vous serais reconnaissant de traiter cette demande en priorité ou de m'indiquer un délai ferme.

{signature_agent}
```
