# Checklist compromis de vente

## Pré-compromis — Documents du vendeur

```
📋 Checklist compromis — ce que tu dois préparer :

Documents de vente (déjà dans ton Drive) :
✅ Renseignements urbanistiques (RU)
✅ Certificat PEB — Score {score}
✅ Rapport contrôle électrique
✅ Attestation de sol
✅ Titre de propriété
{Si Flandre} ✅ Bodemattest + Watertoets + Asbestattest
{Si copro} ✅ Documents syndic (PV AG, charges, fonds de réserve)
{Si citerne} ✅ Attestation citerne mazout

Documents personnels :
☐ Carte d'identité (valide)
☐ État civil (extrait acte de naissance ou livret de famille)
☐ Coordonnées bancaires (IBAN où tu veux recevoir le prix de vente)
☐ Coordonnées complètes de ton notaire

Informations à communiquer au notaire :
☐ Prix de vente convenu : {prix_offre}€
☐ Conditions de la vente (suspensives, délais)
☐ Coordonnées complètes de l'acheteur (nom, adresse, email, téléphone)
☐ Coordonnées du notaire de l'acheteur (si différent)
```

## Pré-compromis — Ce qu'il faut demander à l'acheteur

```
Demande à l'acheteur (ou à son notaire) :

☐ Nom complet + date de naissance
☐ Adresse actuelle
☐ Numéro de registre national
☐ Situation familiale (marié, cohabitant, seul — ça change les droits d'enregistrement)
☐ Nom et coordonnées de son notaire
☐ Confirmation du mode de financement (cash, prêt, montant emprunté)
☐ Délai souhaité pour l'acte authentique
```

## Coordination notaire

### Choix du notaire

```
Tu as déjà un notaire ? Sinon, je peux en recommander un près de chez toi.

En Belgique, chaque partie peut avoir son propre notaire — les honoraires sont partagés, ça ne coûte pas plus cher.
```

### Prise de RDV

```
Appelle ton notaire {nom_notaire} au {telephone} pour fixer la date du compromis.

Dis-lui :
- "Je vends mon bien au {adresse} pour {prix}€"
- "J'ai tous les documents prêts dans un dossier numérique"
- "L'acheteur est {prenom_acheteur}, son notaire est {notaire_acheteur}"
- "On voudrait signer le compromis d'ici {delai} semaines"

Tu veux que je te prépare un email récapitulatif pour le notaire avec tous les documents ?
```

## Post-compromis — Milestones de suivi

| Milestone | Délai | Action |
|-----------|-------|--------|
| **Signature compromis** | J+0 | Confirmer la signature, archiver le compromis signé dans Drive |
| **Délai de réflexion acheteur** | J+0 à J+0 | En Belgique, pas de délai de rétractation légal (sauf clause contractuelle) |
| **Condition suspensive de prêt** | J+0 à J+45 (typique) | Suivi : l'acheteur a-t-il obtenu son prêt ? |
| **Rappel prêt acheteur** | J+30 | "L'acheteur a {X} semaines restantes pour obtenir son prêt. Des nouvelles ?" |
| **Confirmation prêt** | Avant J+45 | Si confirmé → rassurer le vendeur. Si refusé → clause suspensive activée, vente annulée |
| **Préparation acte** | J-14 | "L'acte est dans 2 semaines. Commence à préparer les relevés de compteurs." |
| **Checklist acte** | J-7 | Envoyer la checklist complète (voir closing-checklist.md) |
| **Signature acte** | ~J+120 (4 mois) | Félicitations ! Passer au post-vente |

### Messages de suivi automatiques

**J+7 après compromis :**
```
Le compromis est signé depuis une semaine. L'acheteur a {delai_pret} semaines pour obtenir son prêt bancaire.

Pas d'inquiétude, c'est le temps normal. S'il y a un problème, ton notaire te contactera.
```

**J+30 :**
```
On est à 1 mois du compromis. L'acheteur a-t-il confirmé son prêt ? Si tu n'as pas de nouvelles, tu peux demander à ton notaire de vérifier.
```

**J-14 :**
```
L'acte est dans 2 semaines ! Commence à préparer tes relevés de compteurs (eau, gaz, électricité). Note les chiffres la veille de l'acte.
```

**J-7 :**
```
L'acte est dans 7 jours. Consulte la checklist que je t'envoie maintenant → (voir closing-checklist.md)
```
