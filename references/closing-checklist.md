# Checklist closing — Acte authentique et post-vente

## J-7 : Préparation de l'acte

```
L'acte est dans 7 jours. Voici ce que tu dois préparer :

📋 Checklist acte :

Documents :
☐ Carte d'identité (valide !)
☐ Coordonnées bancaires (IBAN pour recevoir le virement)

Relevés de compteurs (à noter la veille de l'acte) :
☐ Compteur eau — note le chiffre exact
☐ Compteur gaz — note le chiffre exact
☐ Compteur électricité — note le chiffre exact (jour + nuit si bihoraire)

Clés et accès :
☐ Toutes les clés de la porte d'entrée
☐ Clé boîte aux lettres
☐ Clé garage / télécommande portail
☐ Clé cave
☐ Clé local vélo / buanderie (si copro)
☐ Badges d'accès (si immeuble)
☐ Télécommandes (portail, garage, volets)
☐ Code alarme (si applicable)
☐ Code interphone / digicode

Divers :
☐ Certificat de ramonage (si cheminée/poêle)
☐ Notices/garanties appareils laissés dans le bien
☐ Plan du bien (si tu l'as)
```

## Jour de l'acte

```
C'est le grand jour ! Quelques rappels :

🕐 RDV chez le notaire : {date} à {heure}
📍 Adresse : {adresse_notaire}

Ce qui va se passer :
1. Le notaire lit l'acte (ça prend ~30-45 min)
2. Tu signes, l'acheteur signe
3. L'acheteur paie (virement ou chèque certifié)
4. Tu remets les clés à l'acheteur
5. C'est fait ! 🎉

⚠️ N'oublie pas :
- Tes relevés de compteurs (notés ce matin)
- Toutes les clés et télécommandes
- Ta carte d'identité
```

## Post-acte : Résiliations et changements

### Message post-vente principal

```
🎉 C'est fait ! Ta vente est officiellement conclue. Félicitations !

Le virement de {prix}€ arrivera sur ton compte dans les 2-3 jours ouvrables.

Voici les dernières démarches à faire dans les 30 jours :
```

### 1. Assurance habitation

```
📋 Résiliation assurance habitation

Contacte {compagnie_assurance ou "ton assureur"} :
- Par téléphone : {numero}
- Par email : {email}

Dis-leur :
"Je souhaite résilier mon assurance habitation pour le bien situé au {adresse}, vendu le {date_acte}."

Ou envoie cet email :
---
Objet : Résiliation assurance habitation — {adresse}

Madame, Monsieur,

Je vous informe de la vente de mon bien situé au {adresse}, intervenue le {date_acte}.

Je souhaite résilier mon contrat d'assurance habitation n°{numero_contrat si connu} à compter de cette date.

Merci de me confirmer la résiliation et le remboursement du prorata de prime.

Cordialement,
{nom_vendeur}
---
```

### 2. Contrat énergie (gaz + électricité)

```
📋 Transfert/résiliation contrat énergie

Contacte {fournisseur_energie ou "ton fournisseur d'énergie"} :

Donne-leur :
- Adresse du bien vendu
- Date de l'acte : {date_acte}
- Relevés de compteurs du jour de l'acte :
  - Électricité : {releve_elec}
  - Gaz : {releve_gaz}
- Ton numéro de client

Ils s'occupent du reste. La facture de clôture arrivera sous quelques semaines.
```

### 3. Eau

```

Contacte {distributeur_eau ou "ton distributeur d'eau"} :
- Bruxelles : Vivaqua (02 518 82 18)
```

### 4. Internet / téléphonie

```
📋 Résiliation ou transfert internet

Contacte {fournisseur_internet ou "ton fournisseur"} pour :
- Résilier le contrat (attention au préavis, souvent 1 mois)
- OU transférer vers ta nouvelle adresse

N'oublie pas de rendre le modem/routeur si c'est en location.
```

### 5. Autres résiliations possibles

```
☐ Taxe communale (poubelles, etc.) — la commune sera informée automatiquement via le notaire
☐ Précompte immobilier — sera au prorata, géré par le notaire
☐ Abonnement entretien chaudière — résilie si tu en as un
☐ Contrat alarme/télésurveillance — résilie si applicable
☐ Abonnement jardinier/entretien — résilie si applicable
```

## Implications fiscales (guidance uniquement)

```
💰 Côté fiscal :

En Belgique, la vente de ta résidence principale est en principe exonérée de plus-value.

⚠️ Attention si :
- Ce n'est PAS ta résidence principale → possible taxation sur la plus-value
- Tu as acheté il y a moins de 5 ans → risque de taxation comme spéculation immobilière
- Tu as fait des travaux importants → les frais de travaux peuvent réduire la plus-value imposable

Je ne suis pas fiscaliste. Pour toute question fiscale, consulte ton comptable ou ton notaire.
```
